# Animating.Rocks PLUS

An open source Google Chrome Extension to add more features to [Animating.Rocks](https://animating.rocks/).

## Features

- Importing and exporting projects (mp4 exporting coming soon..)
- Share projects with others with a code-based system
- Keybinds (coming soon..)

## Installation Instructions

1. Clone the repository with `git clone https://github.com/SaksornSea/Animating.Rocks-PLUS.git`
2. Go to your browers [extension manager](chrome://extensions/) and enable "Developer mode" (top right corner)
3. Click "Load unpacked" and select the [animating-rocks-plus](animating-rocks-plus/) folder

## Usage

### Importing and Exporting Projects

#### Exporting

1. Click the "Export" button in the top right corner
2. Enter the name of the file (if you want to rename it)
3. Select eather to save as a .arf (Animating.Rocks Project) or .mp4 (mp4 exporting coming soon..)
4. The project will be saved to your Downloads folder

#### Importing

1. Click the "Import" button in the top right corner
2. Select the .arf file you want to import
3. The project will be loaded into the editor

### Sharing Projects

#### Sharing

1. Click the Animating.Rocks PLUS icon in the Chrome toolbar (you might need to pin the extension to the toolbar first)
2. Click the "GENERATE NEW CODE" button to get a code
3. Enter the receiver's code
4. Click the "SHARE" button
5. The project will be sent to the receiver as a request first
6. The receiver will get a notification to accept or decline the request
7. If the receiver accepts the request, the project will be loaded into their editor

#### Receiving

1. Click the Animating.Rocks PLUS icon in the Chrome toolbar (you might need to pin the extension to the toolbar first)
2. Click the "GENERATE NEW CODE" button to get a code
3. Send the code to the sender for them to share the project to you
4. Wait for request to appear in your inbox (you may need to click the "REFRESH" button)
5. Click "Accept" to receive the project
6. The project will be loaded into your editor
